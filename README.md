## Project Overview
Focus Mate is a productivity and study planning application designed to help students stay organized, manage tasks, plan study sessions, and improve focus.

## Features
- User Authentication
- Task Management
- Study Planner
- Productivity Dashboard
- Mood Tracker
- Focus Mode
- Reminder System
